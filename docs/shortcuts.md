# Windows To Mac Shortcut Map

The fastest rule: try Command where you used Ctrl.

| Windows action | Windows shortcut | Mac shortcut |
| --- | --- | --- |
| Copy | Ctrl-C | Command-C |
| Paste | Ctrl-V | Command-V |
| Cut | Ctrl-X | Command-X |
| Undo | Ctrl-Z | Command-Z |
| Redo | Ctrl-Y or Ctrl-Shift-Z | Command-Shift-Z |
| Save | Ctrl-S | Command-S |
| Print | Ctrl-P | Command-P |
| Find | Ctrl-F | Command-F |
| Select all | Ctrl-A | Command-A |
| New window | Ctrl-N | Command-N |
| New tab | Ctrl-T | Command-T |
| Close tab or window | Ctrl-W | Command-W |
| Quit app | Alt-F4 | Command-Q |
| Force quit | Ctrl-Shift-Esc | Option-Command-Esc |
| Switch apps | Alt-Tab | Command-Tab |
| Switch app windows | Alt-Tab or taskbar | Command-Grave |
| Windows-style window switcher | Alt-Tab | AltTab app |
| Lock screen | Windows-L | Control-Command-Q |
| Screenshot region | Windows-Shift-S | Command-Shift-4 |
| Screenshot and recording panel | Snipping Tool | Command-Shift-5 |
| Open settings | Windows-I | System Settings from Apple menu or Spotlight |
| Search apps/files | Windows key | Command-Space |
| Delete file | Delete | Command-Delete |
| Rename file | F2 | Return |
| Open selected file | Enter | Command-Down or Command-O |
| Parent folder | Alt-Up | Command-Up |
| End of line | End | Command-Right |
| Start of line | Home | Command-Left |
| End of document | Ctrl-End | Command-Down |
| Start of document | Ctrl-Home | Command-Up |

## Text Editing Differences

| Goal | Mac shortcut |
| --- | --- |
| Move by word | Option-Left or Option-Right |
| Delete previous word | Option-Delete |
| Delete to end of line | Control-K |
| Paste and match style | Option-Shift-Command-V |
| Emoji and symbols | Control-Command-Space |

## Browser And Tab Habits

Most browser shortcuts move cleanly from Ctrl to Command:

- Command-L focuses the address bar.
- Command-T opens a new tab.
- Command-Shift-T reopens a closed tab.
- Command-Option-Left and Command-Option-Right switch tabs in many apps.

## Terminal Note

Terminal apps use Control for shell commands because Unix predates the Mac shortcut conventions:

- Control-C interrupts a command.
- Control-D sends EOF or exits a shell.
- Control-A and Control-E move to line start and end in many shells.

In terminal windows, Command-C and Command-V still copy and paste through the GUI.
