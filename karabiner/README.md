# Karabiner Examples

These examples are optional starting points for [Karabiner-Elements](https://karabiner-elements.pqrs.org/).

Karabiner can make macOS more comfortable for Windows-trained hands, but it is a deep input tool. Import one rule at a time, test it, and remove what you do not use.

## Examples

- [home-end.json](home-end.json): make Home and End behave like line-start and line-end keys in most Mac apps.
- [caps-lock.json](caps-lock.json): tap Caps Lock for Escape, hold it for Control.
- [external-windows-keyboard.json](external-windows-keyboard.json): global example for swapping Windows-keyboard Alt and Windows keys into Mac-style Option and Command positions.

## Safer Built-In Alternative

For external keyboards, try macOS first:

```text
System Settings > Keyboard > Keyboard Shortcuts > Modifier Keys
```

macOS can remap modifier keys per keyboard. Prefer that if you use a MacBook keyboard and an external Windows keyboard together.

## Importing

1. Open Karabiner-Elements.
2. Go to Complex Modifications.
3. Add your own rule or import a copied JSON file.
4. Enable only the rule you are testing.
5. Keep a normal keyboard path available until you trust the mapping.
