# Security Policy

## Supported Content

This repository is a documentation and shell-script toolkit. Security reports are welcome for:

- Scripts that perform unintended mutations.
- Unsafe shell behavior.
- Incorrect instructions that could expose private data.
- Links to compromised or suspicious downloads.

## Reporting

Please report sensitive issues privately through GitHub Security Advisories if the repository is hosted on GitHub. For non-sensitive issues, open a normal issue with reproduction steps and expected behavior.

## Safety Expectations

Scripts should remain confirm-only and should not pipe remote code into a shell. Installation instructions should prefer official vendor pages, Homebrew Formulae pages, or clearly identified package-manager commands.
